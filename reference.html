Reference Guide

This guide provides a comprehensive overview of all the features available in the Scales Thesaurus application. Use it to understand how to get the most out of our powerful music theory tools.

The Main Application (Interactive Fretboard)

The core of the site is the interactive fretboard page (index.html), where you can visualize and interact with musical patterns.

Core Controls (Page Footer)

The main controls for the fretboard are located in the footer at the bottom of the page. These allow you to define what pattern is displayed.

  • Key: Sets the root note for the scale, chord, or interval.
  • Pattern Type: Choose whether you want to display a Scale, Chord, Interval, or a Grid pattern.
  • Pattern: Select the specific pattern to display (e.g., Major, minor7, Perfect 5th). This list updates based on the Pattern Type.
  • Start/End Fret: Defines the visible range of the fretboard.
  • Display Mode: Changes the text displayed on the notes (e.g., Intervals like R, 3, 5; or Notes like C, E, G).
  • Color/Page Theme & Font Size: Customize the visual appearance of the fretboard and the entire page.

Sidebar Features

The sidebar contains advanced customization and functionality. Click the hamburger icon (☰) in the top header to toggle it.

  • Music Theory: Control which details (like spelling, quality, modes) are shown in the Theory Info Panel below the fretboard.
  • Appearance: A deep dive into visual customization. Here you can change the size, shape, font, and effects of notes, as well as the style of the strings and frets. You can also define your own Custom color theme.
  • Instrument Setup: Choose from presets like guitar or bass, or create your own instrument by setting the number of strings and their individual tunings. You can also control which strings display notes and set unique start/end frets for each string.
  • Chromatic Tuner: A simple tuner with buttons to play any of the 12 chromatic notes.
  • Slideshow: Create a collection of diagrams. Use the "Add to Slideshow" button to add the current fretboard view to your slideshow, which you can then play back, loop, and export.
  • Melodic Patterns: Apply rhythmic and melodic sequences to the currently selected scale for practice. This is a great tool for developing technique and learning scales in a more musical way.
  • Export: Perform bulk exports. You can save diagrams of every scale, chord, or interval for all 12 keys as JPG images, either individually or bundled into a ZIP file for convenience.
  • Pattern Generator: Quickly generate a list of all scales, chords, or intervals for the currently selected key. You can then view them or add them to your slideshow.

Interacting with the Fretboard

  • Clicking a Note: Click on any note marker on the fretboard to open an edit menu. You can change its color, shape, or set it as the new root note for the current pattern. Use Ctrl/Cmd + Click for multi-select and Shift + Click to select all notes of the same type.
  • Action Buttons: Below the fretboard, you'll find buttons to Play Pattern (hear it with the synth), Add to Slideshow, and Export Diagram (save a single image).

Explorer Pages

The Scales, Chords, and Intervals pages provide browsable and filterable lists of all the patterns available in the application.

  • Filtering: Use the filter controls to find patterns that meet specific criteria. For example, on the Scales page, you can find all scales that include a b3 (minor third) but exclude a 7 (major seventh).
  • Favorites: On the Scales page, you can click the star icon (☆) to mark a scale as a favorite. Then, you can use the "Show Favorites Only" filter to see your saved list.
  • View on Fretboard: Each item has a link to open it directly in the main interactive fretboard application for visualization and practice.

Data Management Page

The Data page allows advanced users to view and even edit the core music theory data that powers the application. You can change interval names, scale formulas, etc. Changes are saved to your browser's local storage. You can also export the data tables to XLSX (Excel) files or reset all data back to the application defaults.
